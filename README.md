# muay-share-types

Shared DTO package for Muay services.

## Install

```bash
npm install github:athisan/muay-share-types
```

Or pin a branch:

```bash
npm install github:athisan/muay-share-types#main
```

## Scripts

```bash
npm run build
```

## Usage

```ts
import { JoinQueueDto, LockSeatDto } from 'muay-share-types';
```

## GitHub usage

1. Push this folder to GitHub
2. Install from GitHub with `npm install github:athisan/muay-share-types`
